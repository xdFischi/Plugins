Name: Lobby
Version: 1.0
Author: Alex

Installation:
	1. Plugin in den plugin Ordner ziehen
	2. Server starten
	
Anmerkung:
	Wenn der Server reloaded wird kann es zu bugs kommen, da das Plugin wichtige informationen beim betreten des Servers bekommt.

Kommt noch:
	-/selectswordban <Ban-ID>
	-/selectaxemute <Mute-ID>

	-/adminitems <item>
	-/adminitems clear
	-/adminitems list
	-/build
	-/build <player>
	-/check
	-/check <player>
	-/fly
	-/fly <player>
	-/gm <0|1|2|3>
	-/gm <0|1|2|3> <player>
	-/setgame <1-2>
	-/setspawn
	-/spec

	-/tabreload
